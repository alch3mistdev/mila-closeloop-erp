## Hook Strategy
Peer proof via archetype rather than fabricated logos or testimonials. "Multi-plant manufacturers" signals that peers in similar situations are already thinking about this problem. The three-card narrative arc (Problem → Solution → Result) follows the classic storytelling framework that LinkedIn's own data shows drives higher dwell time on carousel ads. Each card is self-contained but builds on the previous one, encouraging swipe-through.

**Psychological triggers used:**
- **Peer proof by archetype** — "Multi-plant manufacturers" creates implicit social proof without fabrication
- **Narrative arc** — Problem → Solution → Result mirrors the buyer's mental journey
- **Specificity** — "6 plants, 4 legacy systems, 1 target" creates concrete identification
- **Progressive disclosure** — Each card reveals more, rewarding swipe-through
- **Scale signaling** — "50+ location migrations" qualifies the audience and signals enterprise capability

## Creative Brief

### Card 1: "The Problem"
- **Background:** Warm beige (#F1EEE8)
- **Visual:** Scattered system icons/blocks (Oracle, AS/400, Excel, SAP ECC, etc.) with arrows converging toward a single "SAP S/4HANA" block. Visual chaos → single target.
- **Accent:** Teal (#005E66) for labels and icons
- **Typography:** "01" step number in orange, "THE PROBLEM" in mono, body in serif
- **Bottom:** CloseLoop brand mark

### Card 2: "What We Check"
- **Background:** Dark teal (#005E66)
- **Visual:** Four validation checkpoints with check-circle icons in white/orange:
  - ✓ Schema mapping
  - ✓ Data completeness
  - ✓ Value range validation
  - ✓ Cross-plant consistency
- **Typography:** White text, "02" step number in orange, "WHAT WE CHECK" in mono
- **Bottom:** "Automated, not manual." + CloseLoop brand mark in white

### Card 3: "The Result"
- **Background:** Warm beige (#F1EEE8)
- **Visual:** Stylized dashboard showing green pass/fail indicators, remediation list. A "Months → Weeks" timeline compression indicator with orange arrow.
- **Accent:** Orange (#E84E1B) for the compression arrow and step number
- **Typography:** "03" step number in orange, "THE RESULT" in mono, body in serif
- **Bottom:** CloseLoop brand mark

### Design Consistency Across Cards
- All cards use the same typography hierarchy (step number + mono label + serif body)
- Brand mark appears in the same position (bottom-center) on every card
- Color alternation: beige → teal → beige creates visual rhythm during swipe
- Each card has exactly one visual focal point
